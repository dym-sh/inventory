## keyboard shortcuts

### `Meta + KP_x`
- 4 / 6 / 8 / 2
	> move window to the left / right / top / bottom half of the screen

- 1 / 3 / 7 / 9
  > move window to the according corner (1/3rd) of the screen

### `Meta + Space` / `Meta, Meta`
- universal search
- rofi-like
